Highlighted Projects

Adaptive Sampling Variant Calling

Developed a Nextflow-based variant calling pipeline to compare Oxford Nanopore adaptive sampling with whole-genome sequencing for cancer panels. Integrated SNV, SV, and DNA methylation analysis using tools such as Clair3, Sniffles, and modbam2bed, with data visualization in Python and R.

SARS-CoV-2 Wastewater Genomics

Validated Oxford Nanopore sequencing for SARS-CoV-2 variant detection in wastewater samples. Automated genomic processing pipelines using Nextflow and developed algorithms for detecting low-frequency variants.

Plasmodium Genomics

Led genomic and transcriptomic analyses of Plasmodium species to investigate drug resistance, genetic diversity, and host-parasite interactions. Included metagenomics-based microbiome profiling in malaria-affected children.

Multiome ML Integration

Applied machine learning to integrate scRNA-seq and scATAC-seq datasets for cell-type classification and regulatory element prediction. Leveraged dimensionality reduction and clustering techniques to uncover multi-omic patterns.
